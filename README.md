# 5Star Flex/Support Auto-Register

A Chrome extension that automatically signs you up for flex sessions on 5starstudents.com fully spagetticoded :3

---

## What it Does

- Scans upcoming sections
- Finds selected teacher
- Signs up for said teachter
- Skips at capacity teachers
- Logs every action in real time
- Skips and dosent overwirte past sign ups

---

## How to download

1. Click the green **`<> Code`** button at the top of this GitHub page
2. Select **Download ZIP**
3. Once downloaded, unzip the file
4. Open up chrome -> chrome://extensions/ -> Developer Mode -> Load Unpacked
5. Open the extracted folder and find the **`extension`** subfolder inside

---

## How to use

1. Click Extention -> Open 5starstudents
2. Sign in with your Google account
3. Once on the dashboard click the extension
4. Enter your **target teacher's last name** (e.g. `Redmond`) — partial match, case-insensitive
5. Toggle **Tuesday** and/or **Wednesday** on or off as you want
6. Click **Run**
7. Watch the activity log — it will tell you exactly what happened for each date
